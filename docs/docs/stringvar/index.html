    문자열이란 ?¶

    문자열의 길이 가져오기¶

    문자열의 길이를 가져오는 방법은 매우 간단합니다 

    1
2
    local a = "test"
print(#a) -- 4
    다음과 같이 간단하게 앞에 길이 연산자 #을 붇여주면 문자열의 길이를 가져올 수 있습니다

    다만 한글의 경우 바이트 수가 다르기 때문에 한 글자를 3 글자로 인식합니다1 

    1

    문자열과 문자열을 합치기¶

    어떤 문자열과 문자열을 이으려면 .. 연산자를 이용할 수 있습니다2 

    1
    print(..)

    다른 타입을 문자열로 변환하기¶

    기본적으로 lua 에는 tostring 이라는것이 있습니다 이를 이용하면 거의 대부분의 자료형을 문자열로 변환할 수 있습니다 가장 대표적으로 숫자를 문자열로 바꿀 수 있습니다 

    1
    print(type(tostring(1))) -- 예상되는 출력 : 'string'

    함수나 테이블을 변환하는 경우3 해당 값의 고유번호를 가져옵니다4 

    1
2
    print(tostring({}))
-- 이것이 출력하는것은 os 환경, jit 환경 등에 의해 달라질 수 있습니다 따라서 항상 같은 값을 출력하지는 않습니다

    숫자에 한정해서 빈 문자열과 연결하는 것으로 이를 수행할 수도 있습니다5 

    1
    print()

    1. os 환경이나 jit 환경에 따라 달라질 수 있습니다 ↩

    2. 다중 인수로써 사용되는 ... 와 혼동하지 마세요 ↩

    3. 메타테이블은 tostring 동작을 변형시킬 수 있습니다 ↩

    4. 이는 메모리 위치값을 의미합니다 (lua VM) ↩

    5. 권장되는 방법이 아닙니다, 바꾸려는 값이 다른 타입을 가졌다면 오류를 발생시킵니다 ↩
